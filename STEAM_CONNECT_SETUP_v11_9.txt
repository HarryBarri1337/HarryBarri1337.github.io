SkinQuest v11.9 Steam Connect setup

Already needed in Supabase SQL:
1. Run skinquest_upgrade_v11_9_from_v11_8_3.sql.

Already needed in Supabase Edge Functions:
1. Deploy steam-auth-start with --no-verify-jwt.
2. Deploy steam-auth-callback with --no-verify-jwt.
3. Make sure SITE_URL is either set to https://skinquestcs.com or left unset because both functions default to that URL.

Frontend change included in this ZIP:
- Settings > Connected Logins > Steam now calls https://ubvkupqgigfxehprsoit.supabase.co/functions/v1/steam-auth-start.
- Callback redirects back to /settings.html?steam=connected or an error code.

After uploading frontend files to GitHub/Simply:
1. Sign in to SkinQuest.
2. Open Settings.
3. Click Connect on Steam.
4. Sign in through Steam.
5. Confirm that Settings shows Steam as connected.
6. In Supabase Table Editor > profiles, check that steam_id and steam_connected_at are filled on your user row.
